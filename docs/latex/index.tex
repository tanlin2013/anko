Toolkit for performing anomaly detection algorithm on 1D time series based on numpy, scipy.

Conventional approaches that based on statistical analysis have been implemented, with mainly two approaches included\+:
\begin{DoxyEnumerate}
\item Normal Distribution ~\newline
 Data samples are presumably been generated by normal distribution, and therefore anomalous data points can be targeted by analysing the standard deviation. ~\newline

\item Fitting Ansatz ~\newline
 Data samples are fitted by several ansatzs, and in accordance with the residual, anomalous data points can be selected.
\end{DoxyEnumerate}

Regarding model selections, models are adopted dynamically by performing normal test and by computing the (Akaike/\+Bayesian) information criterion. By default, the algorithm will first try to fit in the data into normal distribution, if it passed \href{https://docs.scipy.org/doc/scipy/reference/generated/scipy.stats.normaltest.html}{\texttt{ normal test}}. If this attempt suffers from the loss of convergence or it did not pass \href{https://docs.scipy.org/doc/scipy/reference/generated/scipy.stats.normaltest.html}{\texttt{ normal test}} from begining, then the algorithm will pass data into the second methods and try to execute all the available fitting ansatzs simultaneously. The best fitting ansatz will be selected by information criterion, and finally the algorithm will pick up anomalous points in accordance with the residual. \href{https://tanlin2013.github.io/anko/html/classanko_1_1anomaly__detector_1_1_anomaly_detector.html\#a019359334795d2f05a730ab00085e5e9}{\texttt{ click here to see all available methods.}} ~\newline
